# CNN_For_DR
please install additional package:
1. Tensorflow 2.5.0
2. OpenCV 4.5.2.52	
3. PyQT5 5.9.2

5. klik ok
6. tunggu beberapa saat file akan menampilkan hasil dari preprocessing data dan kemudian menampilkan hasil prediksi di bagian bawah
